## Project Structure

The project structure stems from [text](https://github.com/alan2207/bulletproof-react/blob/master/docs/project-structure.md)

A page has itself a feature folder in the features folder named after it, Each feature folder features

1. api folder // for api calls
2. hooks folder // for reusable functions along the page
3. components folder //
4. types folder
5. screens folder

# This is based on the requirements of the page
